<static> setResponseDecorators(decorators)

Messages, payloads and response are initially simple objects, which later render to JSON.
It is possible to decorate them with your own functions, for example adding more fields.
Every decorator will be invoke for every add (addMessage, addPayload) operation and as a result Response will always have modified objects
Parameters:
Name Type Description
decorators Object
Properties
Name Type Description
response Function
payload Function
message Function
Source:
  • puma.js, line 474
Example
puma.setResponseDecorators({
	response: function (response) {
		response.date = new Date();
		return response;
	},
	payload: function (payload) {
		return payload;
	},
	message: function (message) {
		var customMessage = new CustomMessage(message);
		customMessage.setDate(new Date());
		return customMessage;
	}
});
